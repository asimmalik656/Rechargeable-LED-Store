# Al Saiif Store Website

A complete static ecommerce landing page for the Mini Rechargeable LED Emergency Light.

## Open locally

Double-click `index.html`, or opens the folder in VS Code and use the Live Server extension.

## Important settings

Edit these values in `script.js`:

- `STORE_WHATSAPP`
- `PRODUCT.price`
- `PRODUCT.stock`
- `DELIVERY_CHARGE`

## Replace product images

Put your real JPG images in `assets/images/` using the same filenames, or update the image paths in `script.js` and `index.html`.

## Publish free on Cloudflare Pages

1. Create a GitHub repository.
2. Upload all website files.
3. Sign in to Cloudflare and open **Workers & Pages**.
4. Select **Create application → Pages → Connect to Git**.
5. Select your repository.
6. Use no build command and set the output directory to `/`.
7. Deploy.

## Publish on Netlify

Drag the entire `al_saiif_store` folder into Netlify Drop. Netlify will provide a public link.

## Business checks before publishing

- Replace AI-generated images with real product photos whenever possible.
- Verify charging method, battery details and package contents.
- Confirm your return policy and courier charges.
- Test the WhatsApp checkout on a phone.
